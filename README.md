## Структура проекта

```
game_dungeon/
├── data/
│ ├── armors.json # Данные о броне
│ ├── difficulty.json # Настройки уровней сложности
│ ├── enemies.json # Данные о противниках
│ ├── players.json # Данные о игроках
│ ├── rooms.json # Описания комнат
│ └── weapons.json # Данные об оружии
├── rpg/
│ ├── init.py
│ ├── controller.py # Игровой цикл, ввод/вывод действий игрока
│ ├── combat.py # Логика боя и автобоя
│ ├── dungeon.py # Генерация подземелья по уровню сложности
│ ├── entities.py # Игровые сущности: игрок, враги, комнаты
│ └── loader.py # Загрузка данных из JSON
├── run.py # Запуск игры
├── tests/
│ ├── controller/ # Тесты логики контроллера
│ ├── dungeon/ # Тесты генерации подземелий и комнат
│ └── rpg/ # Тесты игровых сущностей
├── pytest.ini 
└── .gitignore 
```

## Инструкция по запуску

1. Установите pytest для запуска тестов
```bash
python -m pip install pytest
```
2. Запуск игры:

По умолчанию легкий уровень
```bash
python run.py
```

Либо выбрать уровень сложности самостоятельно:
```bash
python run.py lite
```
```bash
python run.py medium
```
```bash
python run.py hard
```

Игровая логика использует input() — вводите номер действия (1,2,...), как описано в контроллере

3. Запуск автотестов:

```bash
pytest -q
```
